# Privacy Policy for Acquia Service Cloud Extension

**Effective Date**: July 30, 2025  
**Version**: 2.0.0

## Overview

The Acquia Service Cloud Extension ("Extension") is designed to enhance your experience with the Acquia Service Cloud platform by providing advanced functionality, performance monitoring, and improved user interface capabilities.

## Information We Collect

### Data Collection
This extension operates with minimal data collection:

- **Website Data**: The extension only accesses and modifies content on `acquia.lightning.force.com` domains as specified in the host permissions
- **Local Storage**: The extension uses Chrome's local storage API to save user preferences and extension settings locally on your device
- **No Personal Data**: We do not collect, store, or transmit any personal information, emails, passwords, or sensitive user data

### Permissions Used
The extension requests the following permissions for legitimate functionality:

- **`activeTab`**: To interact with the current Acquia Service Cloud tab when the extension is actively used
- **`tabs`**: To manage and enhance the Acquia Service Cloud interface
- **`scripting`**: To inject enhancement scripts into Acquia Service Cloud pages
- **`storage`**: To save user preferences and settings locally on your device
- **`host_permissions` for `acquia.lightning.force.com`**: To provide functionality specifically for Acquia Service Cloud

## How We Use Information

All data processing occurs locally on your device. The extension:

- Enhances the Acquia Service Cloud user interface
- Provides performance monitoring tools
- Stores user preferences locally
- Does not transmit any data to external servers
- Does not track user behavior across websites

## Data Sharing

We do not share, sell, rent, or trade any information with third parties. All extension functionality operates locally within your browser.

## Data Security

- All user data is stored locally using Chrome's secure storage APIs
- No data is transmitted over the internet
- Extension operates with minimal required permissions
- Source code follows security best practices

## Data Retention

- Settings and preferences are stored locally until manually cleared by the user
- Uninstalling the extension removes all associated local data
- Users can clear extension data through Chrome's extension settings

## Changes to Privacy Policy

We may update this privacy policy to reflect changes in our practices or for other operational, legal, or regulatory reasons. Users will be notified of any material changes through the extension update process.

## Contact Information

For questions about this privacy policy or the extension:

- **Developer**: Acquia Service Cloud Development Team

## User Rights

Users have the right to:
- Access their locally stored extension data through Chrome's developer tools
- Delete all extension data by uninstalling the extension
- Disable specific extension permissions through Chrome's extension settings

## Compliance

This extension complies with:
- Chrome Web Store Developer Program Policies
- Chrome Extension Manifest V3 requirements
- Data minimization principles

*This privacy policy applies only to the Acquia Service Cloud Extension and does not cover the privacy practices of the Acquia Service Cloud platform itself.*
